Lessons

1 — Python Basics & Syntax

Introduce the interpreter, scripts, indentation rules, printing, and running code.

2 — Variables, Types & Data Structures

Numbers, strings, lists, tuples and dictionaries. Mutability and common operations.

3 — Control Flow & Functions

Conditionals, loops, comprehensions, defining and calling functions, parameters and returns.

4 — Modules, File I/O & Libraries

Importing modules, reading/writing files, and using standard libraries like csv and json.
